## Overview
OpenClaw is a locally-running AI assistant platform with 3,002+ community-built skills. You have full access to install any skill and create specialized agents on demand.

## Quick Commands

### Install Skills
```bash
# Install single skill
npx clawhub@latest install <skill-id>

# Bulk install all skills (takes 10-15 minutes)
python3 /tmp/bulk_installer.py

# Check installed skills
sqlite3 /tmp/kingopus_dashboard.db "SELECT skill_name FROM installed_skills;"
```

### Create Agents
```bash
# List available agents
openclaw agents list

# Create new agent with specific skills
mkdir -p ~/.openclaw/agents/<agent-name>
echo "name: <agent-name>" > ~/.openclaw/agents/<agent-name>/config.json
echo "model: zai/glm-4.7" >> ~/.openclaw/agents/<agent-name>/config.json
```

### Chat with Agents
```bash
# Send message to specific agent
openclaw agent "your message" --agent <agent-id>

# Default agent (deep-researcher)
openclaw agent "your message"
```

## Skill Discovery Workflow

When a user requests a specific capability:

1. **Search Skills**: Use `grep` or check categories above
2. **Install Skill**: `npx clawhub@latest install <skill-id>`
3. **Create Agent**: Make a specialized agent with that skill
4. **Test Agent**: Send a test message to verify

Example:
```bash
# User wants Twitter automation
# 1. Find Twitter skill
grep -i twitter /tmp/all_skills.json

# 2. Install it
npx clawhub@latest install aisa-twitter-api

# 3. Create Twitter agent
mkdir -p ~/.openclaw/agents/twitter-bot
# ... (use template above)

# 4. Test it
openclaw agent "post a tweet about AI" --agent twitter-bot
```
