# Health Journal

Track your food, drinks, symptoms, and notes to identify patterns and triggers.

## Usage

Add entries from command line:
```bash
~/.openclaw/bin/health_journal.py food "Oatmeal with berries"
~/.openclaw/bin/health_journal.py drink "Coffee, black"
~/.openclaw/bin/health_journal.py symptom "Headache" --severity 3
~/.openclaw/bin/health_journal.py note "Feeling tired today"
~/.openclaw/bin/health_journal.py today  # View today's entries
```

Run weekly analysis:
```bash
~/.openclaw/bin/health_analysis.py
```
