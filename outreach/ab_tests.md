## A/B Test Execution Plan

### Phase 1: Setup (Week 1)
1. **Select Test Groups**
   - PM-TEST1: Property management companies with 50+ units
   - PM-TEST2: Property management companies with 100+ units (larger portfolios)
   - AD-TEST3: Auto dealers with 5+ service bays
   - REST-TEST4: Restaurants with $1M+ annual revenue

2. **Prepare Email Lists**
   - Split each group 50/50 between Variation A and B
   - Randomize assignment to avoid bias
   - Ensure equal distribution across geographic markets

3. **Set Up Tracking**
   - Configure email tracking for opens, clicks, replies
   - Create spreadsheet for manual response logging
   - Set up CRM tags for test participants

### Phase 2: Execution (Weeks 2-5)
1. **Send Initial Blast**
   - Send all test emails on Tuesday (highest response day)
   - Send between 9:00-11:00 AM local time
   - Monitor for immediate bounce rates

2. **Follow-Up Schedule**
   - Day 3: First follow-up (reference specific pain point)
   - Day 7: Second follow-up (offer free consultation/assessment)
   - Use same variation for all follow-ups

3. **Daily Monitoring**
   - Track opens, clicks, and responses
   - Log all responses by category (POSITIVE/NEGATIVE/NEUTRAL)
   - Note any unsubscribe or spam complaints

### Phase 3: Analysis (Week 6)
1. **Calculate Metrics**
   - Response rate: (Replies / Sent) √ó 100
   - Meeting rate: (Meetings booked / Sent) √ó 100
   - Open rate: (Opens / Sent) √ó 100
   - Click rate: (Clicks / Sent) √ó 100 (if applicable)

2. **Statistical Significance**
   - Use chi-square test for response rate comparison
   - Calculate p-value (target: p < 0.05 for 95% confidence)
   - Determine confidence interval for each variation

3. **Generate Recommendations**
   - Declare winning variation for each test
   - Document key insights (subject line, opening, value prop, CTA)
   - Propose next test iterations based on findings

### Phase 4: Rollout (Week 7+)
1. **Adopt Winners**
   - Replace original templates with winning variations
   - Update email sequence automation
   - Train outreach team on winning approaches

2. **Next Tests**
   - Test winning elements in new combinations
   - Test follow-up email variations
   - Test personalized vs. generic approaches

### Chi-Square Test Template
```python
# Calculate statistical significance
import scipy.stats as stats

# Example data
responses_a = 12  # Responses in variation A
sent_a = 200
responses_b = 20  # Responses in variation B
sent_b = 200

# Create contingency table
observed = [[responses_a, sent_a - responses_a],
            [responses_b, sent_b - responses_b]]

# Calculate chi-square and p-value
chi2, p_value, dof, expected = stats.chi2_contingency(observed)

if p_value < 0.05:
    print(f"Statistically significant! p={p_value:.4f}")
else:
    print(f"Not significant. p={p_value:.4f}")
```

## Success Criteria

### Minimum Viable Results
- Response rate > 5% (industry average is 1-2%)
- Open rate > 20% (industry average is 15-20%)
- Unsubscribe rate < 1%
- Spam complaint rate < 0.1%

### Winning Variation Criteria
1. **Statistical significance:** p < 0.05
2. **Lift over control:** Minimum 20% improvement
3. **No negative trade-offs:** Higher open rate but much lower quality responses = not a win
4. **Scalable approach:** Can be applied to similar templates/industries

### Iteration Plan
If no clear winner emerges:
- Extend test duration by 2 weeks
- Increase sample size by 50%
- Test new variations combining best elements from A and B
- Consider A/B/C testing with a third variation

## Notes & Considerations

### External Factors
- **Seasonality:** Avoid testing during holiday periods or industry-specific slow seasons
- **Market conditions:** Economic downturns may make cost-focused angles more effective
- **Competitor activity:** Sudden competitor campaigns may affect response rates

### Email Deliverability
- Maintain consistent sending volume and schedule
- Monitor sender reputation scores
- Warm up new domains before bulk sending
- Keep bounce rate below 2%

### Ethics & Compliance
- Honor unsubscribe requests immediately
- Don't misrepresent affiliation with competitors
- Be transparent about commission structure when asked
- Follow CAN-SPAM Act guidelines (physical address, opt-out link)
