# OpenClaw Quick Reference

## Essential Commands
- `openclaw agent "message"` - Chat with default agent
- `npx clawhub@latest install <skill>` - Install skill
- `openclaw agents list` - List agents

2. Install with `npx clawhub install`
3. Create agent directory
4. Add config.json and prompt.md
5. Test with `openclaw agent`
